# Kemet Journey - Jira Epic Separation

This folder separates the extracted `kemet-journey-export` project according to the Jira screenshots you sent.

The original runnable project is still in:

`../kemet-journey-export`

This separated folder is for planning, GitHub branches, and Jira review. I did not move files out of the original project because `index.html`, `app.js`, `styles.css`, and the Java service files share logic across multiple features.
